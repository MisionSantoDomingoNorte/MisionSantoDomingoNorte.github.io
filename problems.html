Tech Problems

Some Helpful hints for when people run into problems

🛠️ Tech Responsibilities You May Encounter

Aside from your daily tasks, you’ll often be the go-to for solving random tech issues that come up for missionaries. Here’s a breakdown of the most common situations and how to handle them like a pro:

💬 1. WhatsApp Issues

Missionaries will definitely run into WhatsApp problems. Here’s how to handle the most frequent ones:

🚫 Blocked for Spam

  • Not much you can do.
  • Have them submit a review request through WhatsApp and wait for the ban to lift.

🔐 Not Receiving Verification Codes

  • Make sure the default messaging app is set to Google Messages:
    • Settings > Messages > Messaging Application
    • Choose the app with a white square and blue message bubble.
  • If the SMS still doesn’t arrive:
    • Try the call verification instead.
  • If both fail: they may just need to wait it out.

🔄 Switching WhatsApp Accounts

Direct them to the WhatsApp Change Guide you’ve created for step-by-step help.

📧 2. Area Email & PMG App Issues

These always pop up after transfers—especially with missionaries going into new areas.

  • Area email and PMG access usually take 1–2 days to update after the transfer.
  • If missionaries are whitewashing, they won’t have access to the previous WhatsApp backup.
    ➤ Nothing can be done to speed this up—just reassure them and have them wait it out.

🗺️ 3. Newly Created Areas

If a new area gets set up, there are a few key things you’ll need to check on:

🧭 Teaching Area Setup

  • Is active in the PMG app
  • Can report key indicators
  • Is set to receive referrals

🛑 Boundary Adjustments

  • Work with the APs to verify the boundaries are correct.
  • Note: Boundaries are locked for editing for about 3 days after a new area is created. After that, you can go in and adjust them as needed.
